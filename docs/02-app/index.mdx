应用路由器是一个使用React最新特性构建应用的新范例。
如果你已经熟悉Next.js，你会发现App Router是基于[Pages Router](/docs/pages)中现有的基于文件系统的路由器的自然进化。

对于新应用，我们建议使用应用路由器。对于现有的应用，你可以[增量迁移到应用路由器](/docs/app/building-your-application/upgrading/app-router-migration)。
