:::caution **废弃**

这个选项在Next.js 13.4中不再需要。App Router现在稳定了。

:::

应用路由器([`app` 目录](/docs/app/building-your-application/routing)) 支持[布局](/docs/app/building-your-application/routing/pages-and-layouts), [服务器组件](/docs/getting-started/react-essentials#server-components), [流](/docs/app/building-your-application/routing/loading-ui-and-streaming), 和[同步数据获取](/docs/app/building-your-application/data-fetching).

使用`app`目录会自动启用[React严格模式](https://react.dev/reference/react/StrictMode).
学习如何[逐步采用`app`](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app).
