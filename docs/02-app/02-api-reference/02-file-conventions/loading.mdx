一个**加载**文件可以创建即时加载状态，建立在[悬念](/docs/app/building-your-application/routing/loading-ui-and-streaming).

默认情况下，该文件是一个[服务器组件](/docs/getting-started/react-essentials#server-components)，但也可以通过`use Client`指令作为客户端组件使用。

```tsx title="app/feed/loading.tsx" switcher
export default function Loading() {
  // Or a custom loading skeleton component
  return <p>'Loading...'</p>
}
```

```jsx title="app/feed/loading.js" switcher
export default function Loading() {
  // Or a custom loading skeleton component
  return <p>'Loading...'</p>
}
```

加载UI组件不接受任何参数。

> **注意事项**
>
> - 在设计加载UI时，你可能会发现使用[React开发工具](https://react.dev/learn/react-developer-tools)手动切换悬念边界很有帮助。
