Next.js 提供了构建块来创建灵活的全栈web应用程序。构建应用程序中的指南解释了如何使用这些功能以及如何自定义应用程序的行为。

章节和页面是按顺序组织的，从基本到高级，因此在构建Next.js应用程序时可以一步一步地遵循它们。但是，您可以按任何顺序阅读它们，或者跳到适用于您的用例的页面。

<AppOnly>

如果你是Next.js的新手，我们建议从[Routing](/docs/app/building-your-application/routing)，[Rendering](/docs/app/building-your-application/rendering)，[Data抓取](/docs/app/building-your-application/data-fetching)和[Styling](/docs/app/building-your-application/styling)部分开始，因为它们介绍了基本的Next.js和web概念来帮助你入门。
然后，你可以深入到其他部分，如[优化](/docs/app/building-your-application/optimizing) 和[配置](/docs/app/building-your-application/configuring)。
最后，一旦你准备好了，签出[部署](/docs/app/building-your-application/deploying)和[升级](/docs/app/building-your-application/upgrading)部分。

</AppOnly>

<PagesOnly>

</PagesOnly>
