[Turbopack](https://turbo.build/pack)(测试版)是一个针对JavaScript和TypeScript优化的增量打包器，用Rust编写，内建在Next.js中。

## 使用

Turbopack can be used in Next.js in both the `pages` and `app` directories for faster local development.
To enable Turbopack, use the `--turbo` flag when running the Next.js development server.

```json title="package.json" highlight={3}
{
  "scripts": {
    "dev": "next dev --turbo",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}
```

## 支持的功能

要了解有关Turbopack当前支持的特性的更多信息，请查看[文档](https://turbo.build/pack/docs/features)。

## 不支持的功能

Turbopack目前只支持`next dev`，不支持`next build`。我们目前正在为构建提供支持，因为我们离稳定更近了一步。
