> 用于检测 _click_ / _tap_ / _point_ 和 _hold_ 事件的React钩子

# 主要特点

- 鼠标，触摸和指针事件支持
- 传递自定义上下文并在回调中访问它
- 如果移动得离目标太远，取消长按
- 灵活的回调: `onStart`, `onMove`, `onFinish`, `onCancel`
- 必要时禁用钩子
- 过滤不需要的事件(如鼠标右键)
