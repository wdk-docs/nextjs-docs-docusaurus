# 按完成

当按压元件完成时

## 名字

onFinish

## 类型

```ts
(event, meta) => void
(
event:
| ReactMouseEvent<Target>
| ReactTouchEvent<Target>
| ReactPointerEvent<Target>,
meta: { context?: Context; reason?: LongPressCallbackReason }
) => void
```

## 默认值

undefined

## 描述

当元素上的按被释放时调用，这总是在一段时间之后，因此在长按发生后调用。
这个回调与callback是互斥的，这意味着如果长按在被触发之前被取消，它将不会被调用。

如果你想在用户释放长按后触发动作，而不是在时间过去后立即触发动作，使用这个回调而不是长按回调
