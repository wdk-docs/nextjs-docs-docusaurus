# 指南

这些指南解释了如何在MDX周围完成几个常见的用例和模式。

1.  GitHub flavored markdown (GFM)

    本指南探讨了如何支持GFM特性，如自动链接文字、脚注、删除、表格和任务列表。

2.  前页

    本指南探讨了如何在MDX中支持YAML前端内容。

3.  数学

    本指南探讨了如何在MDX中支持数学(LaTeX)。

4.  语法高亮显示

    本指南探讨如何将语法高亮显示应用于代码块。

5.  嵌入

    本指南探讨了如何在markdown中嵌入`tweets`、`gist`或`codependencies`等内容。

6.  MDX on demand

    本指南展示了如何使用`@mdx-js/mdx`在服务器上编译`mdx`并在客户端上运行结果。
