# 包

这个索引列出了在我们的单线程中维护的包。
它们包括`@mdx-js/mdx`，这是我们的核心编译器;`remark-mdx`，这是支持MDX语法的remark插件;以及一些与捆绑器和前端框架的集成。

1. [@mdx-js/esbuild](./esbuild)  
   MDX的esbuild插件。

2. [@mdx-js/loader](./loader)  
   MDX的webpack加载器。

3. [@mdx-js/mdx](mdx)  
   MDX编译器。

4. [@mdx-js/node-loader](node-loader)  
   用于MDX的Node加载器。

5. [@mdx-js/preact](preact)  
   MDX的Preact上下文.

6. [@mdx-js/react](react)  
   MDX的React上下文。

7. [@mdx-js/register](register)  
   需要MDX的Node挂钩。

8. [@mdx-js/rollup](rollup)  
   MDX的Rollup插件。

9. [@mdx-js/vue](vue)  
   MDX的Vue上下文。

10. [remark-mdx](remark-mdx)  
    remark插件支持MDX语法(JSX，表达式，导入/导出)。
